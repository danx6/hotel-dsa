# Hotel DSA

Hotel reservation system developed in Python for the Data Structures & Algorithms (DSA) assignment.

## Requirements

- Python 3.x
- Visual Studio Code (recommended)

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/danx6/hotel-dsa.git
   ```

2. Open the project folder:
   ```bash
   cd hotel-dsa
   ```

3. Run the program:
   **Windows**
   ```bash
   python program.py
   ```

   **macOS / Linux**
   ```bash
   python3 program.py
   ```
